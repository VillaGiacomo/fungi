# Fungi AI

PWA sperimentale per mappe porcini, profili locali, aggiornamento meteo e archivio personale delle fungaie.

I punti e le impostazioni personali sono salvati esclusivamente nel browser del singolo dispositivo. Il repository non contiene punti fungaia degli utenti.

## Installazione su iPhone

Apri il sito con Safari, tocca **Condividi** e scegli **Aggiungi alla schermata Home**.
